# Project Overview

This project analyzes video game sales data to understand which games, platforms, genres, and regions perform the best globally. The analysis is done completely using Microsoft Excel, focusing on clear insights and easy-to-understand visuals.

The dataset contains information about popular video games and their sales across different regions.

# Key Columns:

Rank – Global ranking based on total sales

Name – Name of the video game

Platform – Console or system the game was released on

Year – Release year of the game

Genre – Category of the game (Sports, Racing, RPG, etc.)

Publisher – Company that published the game

NA_Sales – Sales in North America (in millions)

EU_Sales – Sales in Europe (in millions)

JP_Sales – Sales in Japan (in millions)


# Objectives of the Project

1. Identify top-selling video games worldwide

2. Compare regional sales (NA, EU, Japan, Others)

3. Find which platforms and genres are most successful

4. Analyze publisher performance
